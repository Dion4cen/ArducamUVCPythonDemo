# arducam uvc python demo
## introduce
arducam camera opencv python demo
## install dependencies
```shell
python -m pip install -r requirements.txt
```
## RUN 
```shell
python arducam_demo.py -W 3840 -H 2160 -f 20 -d 1280:720
```

If you want to capture video, please run the following command:
```
python3 arducam_thread_rec.py -w 1920 -h 1080 -f 30
```
